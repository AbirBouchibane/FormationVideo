## Commandes à éviter

Ces commandes ne sont pas sécurisées, plus maintenues et/ou devenues obsolètes :

+ arp
+ ftp
+ ifconfig
+ ifdown
+ ifup
+ iptunnel
+ iwconfig
+ nameif
+ netstat
+ rcp
+ route
+ telnet
